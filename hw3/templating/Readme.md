# Full Stack HW 3

Option 3 - Templating Exercise

## Description

Uses Express.js, Pug, and an API to display data on a website running on local host

## Getting Started

### Dependencies

- See package-lock.json and package.json files

### Executing program

- in the "hw3/templating/" directory, run:

```
node app.js
```

- Then navigate to http://localhost:5001
